# UDBank_Frontend

## Requisitos

### Node 10

## Instalación

1. Ejecutar ```npm install```

## Lanzar el frontend

### Producción

1. Ejecutar ```npm run build```
2. Ejecutar ```npm start```

Navegar a localhost:5000 o 192.168.0.X:5000

### Debug

Ejecutar ```npm run deploy```

Navegar a localhost:4200
